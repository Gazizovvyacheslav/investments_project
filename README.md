## 1. Цель практической части
Цель — построить и проверить модель оптимального портфеля на основе методов курса (mean-variance, efficient frontier, CAPM, VaR/ES). Проверяется, даёт ли такая стратегия лучшее соотношение риск/доходность по сравнению с бенчмарками.

## 2. Постановка задачи
Нужно выбрать веса активов $w$, максимизирующие доходность при ограничении риска, и проверить устойчивость решения через rolling backtest. То есть построить воспроизводимую стратегию с регулярным пересчётом весов.

## 3. Связь с курсом
Используются:
– $E(r_p)\to\max$ при $\sigma(r_p)\le c$  
– $\sigma(r_p)\to\min$ при фиксированной доходности  
– эффективная граница  
– CML и тангенциальный портфель  
– CAPM ($\beta$)  
– VaR и Expected Shortfall  

## 4. Данные
ETF: SPY, QQQ, IWM, XLF, XLK, XLE, XLV, TLT, GLD.  
Период ~5 лет, дневные данные (yfinance).  
Безрисковая ставка — ^IRX, приведённая к дневной.

## 5. Подготовка данных
Синхронизация дат, заполнение пропусков, расчёт доходностей:
$r_{i,t} = P_{i,t}/P_{i,t-1} - 1$.

## 6. Метрики
Доходность: $E(r_p)=w^T\mu$  
Риск: $\sigma_p=\sqrt{w^T\Sigma w}$  
Дополнительно: VaR, ES, $\beta$ к SPY.

## 7. Оптимизация
Основная задача:
$\max w^T\mu$ при $\sigma\le c$, $\sum w_i=1$, $w_i\ge0$.

Также рассматривался вариант с короткими продажами.  
Эффективная граница — через минимизацию риска при заданной доходности.  
Тангенциальный портфель — максимум Sharpe.

## 8. Калибровка
Rolling окна (126–504 дней), ребалансировка monthly/quartely.  
Параметры выбирались по Sharpe с контролем ES и просадки.

## 9. Бэктест
Walk-forward без look-ahead:  
на каждой дате используются только прошлые данные, веса применяются вперёд.

## 10. Бенчмарки
– Equal Weight  
– SPY  

## 11. Метрики сравнения
Доходность, волатильность, Sharpe, max drawdown, VaR, ES, $\beta$, RAROC.

## 12. Реализация
Python: pandas, numpy, scipy.optimize, matplotlib, yfinance.

## 13. Результаты
Оптимизационная стратегия даёт лучшее соотношение риск/доходность по сравнению с бенчмарками.  
No-short вариант оказался стабильнее по Sharpe, просадке и ES.

## 14. Ограничения
– зависимость от истории  
– чувствительность к параметрам  
– нет транзакционных издержек  
– упрощающие предпосылки модели  

## 15. Вывод
Методы из курса (mean-variance, CAPM, VaR/ES) реально работают на практике и позволяют построить конкурентную инвестиционную стратегию.
